## Start Project Quickly Desktop App 🚴🏻‍♀🏄🏻‍♂️🏇🏻
It makes it possible to launch multi projects with one click 👆🏼👆🏼👆🏼
### Installation
```
npm install
```

### Run
```
npm start
```

### Build Desktop App
You can config app information in package.json
```
npm run dist
```
